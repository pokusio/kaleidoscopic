RGB and Opacity of the Background Color of Kaleidoscopic

Use the inputs below to reset the background color in this page

Using the REST API Endpoint you can also reset the color try 

            
            export JSON_PAYLOAD="{ \
            \"red\" : \"<0 to 255>\", \
            \"green\" : \"<0 to 255>\", \
            \"blue\" : \"<0 to 255>\", \
            \"opacity\" : \"<decimal from 0 to 1>\" \
            }"

            echo "${JSON_PAYLOAD}" | jq .

            curl -d  "${JSON_PAYLOAD}" -X POST http://${KALEIDOSCOPIC_API_HOSTNAME}:${KALEIDOSCOPIC_API_PORT_NO}/api/v1/files/management/ui/theme/maincolor | jq .
Red Component (An integer between 0-255)
Green Component (An integer between 0-255)
Blue Component (An integer between 0-255)
Opacity (A decimal number between 0 and 1)
